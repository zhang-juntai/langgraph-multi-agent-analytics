# 数据分布分析

## 功能概述

分析数值特征的分布形态，包括：
- 生成直方图可视化分布
- 计算偏度（skewness）- 衡量分布的对称性
- 计算峰度（kurtosis）- 衡量分布的尾部厚度

## 参数说明

本技能不需要额外参数，自动检测所有数值列进行分析。

## 输出内容

### 1. 分布直方图
- 为每个数值列生成直方图
- 多子图布局（3列网格）
- 自动隐藏多余子图

### 2. 偏度和峰度
- **偏度 (Skewness)**:
  - 偏度 = 0: 完全对称分布（如正态分布）
  - 偏度 > 0: 右偏（正偏），长尾在右侧
  - 偏度 < 0: 左偏（负偏），长尾在左侧
  - |偏度| > 1: 高度偏态
  - 0.5 < |偏度| < 1: 中度偏态
  - |偏度| < 0.5: 近似对称

- **峰度 (Kurtosis)**:
  - 峰度 = 0: 正态分布的尾部厚度（超出峰度）
  - 峰度 > 0: 厚尾（heavy-tailed），极端值较多
  - 峰度 < 0: 薄尾（light-tailed），极端值较少

## 使用场景

- 数据探索和特征理解
- 检测正态分布假设的违反
- 识别需要变换的偏态特征
- 异常值检测的前期分析

## 示例输出

```
【偏度和峰度】
  age: 偏度=0.234, 峰度=-0.567
  salary: 偏度=1.234, 峰度=2.345
  score: 偏度=-0.123, 峰度=0.456
```

## 统计学解释

### 偏度的实际意义
- **收入分布**: 通常右偏（少数高收入者拉长右尾）
- **考试成绩**: 可能左偏（高分学生较少）
- **年龄分布**: 可能对称或右偏（取决于人群）

### 峰度的实际意义
- **金融回报**: 通常厚尾（极端事件比正态分布预测的更多）
- **测量误差**: 通常薄尾（集中在均值附近）
- **网络流量**: 可能厚尾（偶尔的流量峰值）
